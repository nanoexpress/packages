# @nanoexpress/bun-core

nanoexpress Core for Bun runtime

## Installation

```bash
npm install @nanoexpress/bun-core
# or
yarn install @nanoexpress/bun-core
# or
pnpm install @nanoexpress/bun-core
# or
bun add @nanoexpress/bun-core
```
